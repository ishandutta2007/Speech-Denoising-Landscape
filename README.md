## 🔍 Overview
**Speech Denoising** is the critical process of isolating human speech from background interference. This repository tracks the rapidly evolving landscape of **Neural Speech Enhancement**, **Dereverberation**, and **Real-time Audio Cleaning**.

## 🤖 SOTA Models from GitHub
The following implementations represent the pinnacle of open-source **Speech Enhancement**.

| Model | Resources | Key Innovation | Quality |
| :--- | :--- | :--- | :--- |
| **MP-SENet** | [🔗 Code](https://github.com/yxlu-0102/MP-SENet) / [📄 Paper](https://arxiv.org/abs/2305.13620) | Parallel Magnitude/Phase Denoising | 🌟 A++ |
| **CMGAN** | [🔗 Code](https://github.com/ruizhecao96/CMGAN) / [📄 Paper](https://arxiv.org/abs/2209.11112) | Conformer + Metric GAN | 🌟 A++ |
| **DeepFilterNet** | [🔗 Code](https://github.com/Rikorose/DeepFilterNet) | Real-time Deep Filtering (48kHz) | 🌟 A+ |
| **UVR (Ultimate Vocal Remover)** | [🔗 Code](https://github.com/Anjok07/ultimatevocalremovergui) | Multi-algorithm GUI for separation | 🌟 A++ |
| **noisereduce** | [🔗 Code](https://github.com/timsainb/noisereduce) | Simple Spectral Gating | 🌟 B+ |
| **Wavenet Denoising** | [🔗 Code](https://github.com/drethage/speech-denoising-wavenet) | Autoregressive Wavenet | 🌟 A |
